## Coding Guidelines

- Write clean, readable code.
- Use meaningful variable names.
- Keep functions focused.

## Additional Project Rules

1. Every form must include complete client-side validation before submission.

2. Every input element must have an associated label to improve accessibility.

3. Ask the AI to review and verify generated code before accepting the implementation.

4. Test all generated features manually before committing.

5. Use semantic HTML and responsive design for all user interfaces.
